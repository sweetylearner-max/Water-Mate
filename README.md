# 💧 Water Mate - Your Personal Water Reminder App

Water Mate is a beautifully crafted water reminder app built with **React Native**. It's not just a hydration tracker — it's your **personal health companion** with a modern, delightful user experience that keeps you motivated to stay hydrated every single day.

---

## ✨ Features

- 🌊 **Personalized Water Reminders** — Customizable intervals and quantity tracking based on your schedule and goals
- 📱 **Modern UI/UX** — Sleek, clean interface with intuitive interactions and a minimalist premium design
- 🧠 **Smooth Onboarding Flow** — User-friendly onboarding with contextual hints
- 🔔 **Engaging Notifications** — Built with [Notifee](https://notifee.app/) for rich local push notifications with action buttons, sounds, and deep links
- 🌗 **Dark Mode Support** — Automatically adapts to system theme or toggle manually
- 📊 **Hydration Stats** — Visual charts to track weekly/monthly intake with goal progress and motivational insights
- 👤 **Custom Avatars & Profiles** — Personalize your experience with user avatars
- 💥 **Smooth Animations** — Micro-interactions and transitions using React Native Reanimated, React Native SVG, and Lottie
- 📂 **Android Support** — Currently supports Android (iOS coming soon)

## 📥 Installation

### Prerequisites
- Node.js >= 18
- Android Studio + Android SDK
- React Native CLI

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/sweetylearner-max/Water-Mate.git
   cd Water-Mate
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run on Android**
   ```bash
   npx react-native run-android
   ```

4. **Start Metro bundler** (if not auto-started)
   ```bash
   npx react-native start
   ```
